# 🏏 IPL 2026 Prediction System

A full-stack ML-powered system that predicts IPL match outcomes and tournament probabilities in real time.

## 🚀 Features
- Ensemble ML Model (XGBoost + RF + Logistic Regression)
- Monte Carlo Tournament Simulation (10,000 runs)
- Live Toss Impact Prediction
- Automated Data Pipeline (GitHub Actions)
- Real-time Dashboard (Streamlit)
